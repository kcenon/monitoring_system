Detailed Description

File descriptor usage monitoring collector.

Collects file descriptor usage metrics with cross-platform support. Gracefully degrades when certain metrics are not available on the current platform (e.g., system-wide FD count on macOS/Windows).

Uses CRTP base class to reduce code duplication.
