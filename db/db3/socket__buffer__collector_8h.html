Detailed Description

Socket buffer usage monitoring collector.

This file provides socket buffer (send/receive queue) usage monitoring using platform-specific APIs. Tracking socket buffer fill levels helps detect network bottlenecks, slow connections, and dropped packets at the socket level.

Platform APIs:

  • Linux: /proc/net/tcp (tx_queue, rx_queue), /proc/net/sockstat
  • macOS: netstat -m (mbuf statistics), sysctl kern.ipc
  • Windows: GetTcpStatistics() API (stub implementation)

This is part of the public API. Stable interface - breaking changes follow semantic versioning.

Note
For C++20 module migration:
  • This will be exported as part of kcenon.monitoring.collectors
  • Public classes: socket_buffer_collector, socket_buffer_metrics
