Detailed Description

Registration of built-in metric collectors with the factory.

This file provides a single function to register all built-in collectors with the metric_factory. Call register_builtin_collectors() once at application startup to enable factory-based collector creation.

Usage:

#include <kcenon/monitoring/factory/builtin_collectors.h>
int main() {
kcenon::monitoring::register_builtin_collectors();
auto& factory = kcenon::monitoring::metric_factory::instance();
auto collector = factory.create("system_resource_collector", {});
}
